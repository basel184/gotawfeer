# برومت تحديد المنتجات التي تدعم التجربة الافتراضية

## 📋 الملخص
هذا البرومت موجه للرافيل (مسؤول لوحة التحكم) لتحديد وإضافة المنتجات التي تدعم ميزة التجربة الافتراضية (Virtual Try-On) للمكياج.

---

## 🎯 الفئات المدعومة حالياً

### 1. **المكياج الأساسي (Category ID: 2410)**
- **كريم الأساس / Foundation** (Sub-category: 2427)
  - الوصف: منتجات تغطية الوجه والأساس
  - الاستخدام: تجربة درجات الأساس المختلفة على الوجه

- **الكونتور / Contour** (Sub-category: 2438)
  - الوصف: منتجات النحت والتحديد
  - الاستخدام: تجربة تأثيرات النحت على الوجه

- **الكونسيلر / Concealer** (Sub-category: 2428)
  - الوصف: منتجات إخفاء العيوب
  - الاستخدام: تجربة الكونسيلر تحت العينين

- **أحمر الخدود / Blush** (Sub-category: 2431)
  - الوصف: منتجات تلوين الخدود
  - الاستخدام: تجربة درجات البلاشر المختلفة

### 2. **الشفاه (Category ID: 2415)**
- **أحمر الشفاه / Lipstick** (Sub-category: 2496, 2501)
  - الوصف: أحمر الشفاه بأنواعه (صلب وسائل)
  - الاستخدام: تجربة درجات أحمر الشفاه على الشفاه

- **محدد الشفاه / Lip Liner** (Sub-category: 2497)
  - الوصف: أقلام تحديد الشفاه
  - الاستخدام: تجربة محددات الشفاه

### 3. **العيون (Category ID: 2412)**
- **ظلال العيون / Eye Shadow** (Sub-category: 2464)
  - الوصف: باليتات وألوان ظلال العيون
  - الاستخدام: تجربة درجات الظلال على الجفون

- **الكحل / Eye Liner** (Sub-category: 2463)
  - الوصف: أقلام وسوائل الكحل
  - الاستخدام: تجربة أنماط الكحل المختلفة

- **الماسكارا / Mascara** (Sub-category: 2517)
  - الوصف: منتجات الرموش والماسكارا
  - الاستخدام: تجربة تأثيرات الرموش

- **أقلام الحواجب / Eyebrow Pencil** (Sub-category: 2461)
  - الوصف: أقلام وألوان الحواجب
  - الاستخدام: تجربة درجات الحواجب

## 📝 متطلبات المنتج

لكي يدعم المنتج التجربة الافتراضية، يجب أن يتوفر:

### 1. **الألوان (Colors)**
```json
{
  "colors_formatted": [
    {
      "id": 1,
      "name": "اسم اللون",
      "code": "#RRGGBB",  // كود اللون بصيغة HEX
      "hexCode": "#RRGGBB"
    }
  ]
}
```

### 2. **المتغيرات (Variations)**
```json
{
  "variation": [
    {
      "id": 1,
      "type": "اسم المتغير",
      "qty": 10  // الكمية المتاحة (يجب أن تكون > 0)
    }
  ]
}
```

### 3. **الفئة والفئة الفرعية**
- يجب أن ينتمي المنتج إلى أحد الفئات المدعومة أعلاه
- يجب تعيين الفئة الفرعية الصحيحة

## 🔧 خطوات الإضافة في لوحة التحكم

### الخطوة 1: اختيار الفئة الصحيحة
1. اذهب إلى إدارة المنتجات
2. اختر المنتج أو أنشئ منتج جديد
3. تأكد من اختيار الفئة من القائمة أعلاه:
   - **2410** (المكياج الأساسي)
   - **2415** (الشفاه)
   - **2412** (العيون)

### الخطوة 2: اختيار الفئة الفرعية
اختر الفئة الفرعية المناسبة:
- للأساس: **2427**
- للكونتور: **2438**
- للكونسيلر: **2428**
- للبلاشر: **2431**
- لأحمر الشفاه: **2496** أو **2501**
- لمحدد الشفاه: **2497**
- لظلال العيون: **2464**
- للكحل: **2463**
- للماسكارا: **2517**
- لأقلام الحواجب: **2461**

### الخطوة 3: إضافة الألوان
1. في قسم "الألوان" (Colors):
   - أضف كل لون متاح للمنتج
   - تأكد من إدخال كود اللون بصيغة HEX (#RRGGBB)
   - أضف اسم اللون واضح (مثل: أحمر ملكي، وردي فاتح، إلخ)

### الخطوة 4: إضافة المتغيرات
1. في قسم "المتغيرات" (Variations):
   - أضف كل متغير متاح (مثل الحجم أو النوع)
   - تأكد من إدخال الكمية المتاحة (qty > 0)
   - اربط كل متغير باللون المناسب إن أمكن

### الخطوة 5: حفظ المنتج
- تأكد من حفظ جميع البيانات
- تحقق من أن المنتج ظهر في الموقع

## ✅ قائمة التحقق

قبل نشر المنتج، تأكد من:

- [ ] المنتج في الفئة الصحيحة (2410, 2415, أو 2412)
- [ ] الفئة الفرعية صحيحة
- [ ] تم إضافة الألوان بصيغة HEX صحيحة
- [ ] كل لون له اسم واضح
- [ ] تم إضافة المتغيرات مع الكميات
- [ ] الكمية المتاحة > 0
- [ ] تم حفظ المنتج بنجاح
- [ ] المنتج يظهر في صفحة المنتج مع زر "جربي المنتج افتراضياً"

## 🎨 نصائح مهمة

1. **اختيار الألوان الدقيقة**
   - استخدم أداة اختيار الألوان لضمان دقة كود HEX
   - تأكد من أن الألوان تطابق المنتج الفعلي

2. **تسمية الألوان**
   - استخدم أسماء وصفية وسهلة الفهم
   - تجنب الأسماء الغامضة

3. **الكميات**
   - تأكد من تحديث الكميات بانتظام
   - لا تضف منتجات بكمية صفر

4. **الصور**
   - أضف صور عالية الجودة للمنتج
   - يفضل صور توضح الألوان بوضوح
